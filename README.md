# 펭귄 시소 게임
## 1. 개요
떨어지는 펭귄의 위치를 조절해 시소의 균형을 맞추는 게임이다. 균형이 맞지 않으면 무거운 쪽으로 시소가 기울고, 일정 시간 내에 균형을 맞추지 못하면 펭귄이 굴러 떨어져 게임이 끝난다. 바람 소리와 펭귄 울음소리로 생동감을 더해 게임에 더욱 몰입할 수 있도록 한다.
## 2. 게임에 대한 기본 설명
- 화살표키, 혹은 A,D 키로 펭귄의 좌우 위치를 조절한다.
- 아래방향 화살표키 혹은 S키를 누르면 펭귄이 더 빠르게 떨어지고, 추가 점수를 얻는다.
- 시소의 균형이 맞지 않으면 시소가 무거운 쪽으로 기울고, 펭귄들도 시소를 따라 미끄러진다.
- 일정 시간동안 시소의 균형을 맞추지 못하거나 펭귄이 시소 밖으로 떨어지면 게임이 끝나기때문에 적당한 위치에 펭귄을 빠르게 착지시켜 시소의 균형을 맞춰야 한다.
## 3. 주요 수정 사항
1) 펭귄이 시소의 한가운데로 떨어져서 아무것도 하지 않으면 시소가 그대로 있는 문제가 있었음. -> 펭귄이 떨어지기 시작하는 지점을 랜덤으로 설정함.
2) 펭귄이 시소 밖으로 떨어져도 게임이 끝나지 않음. -> 펭귄이 시소 밖으로 떨어지면 게임이 끝나도록 수정함.
3) 떨어지는 펭귄이 시소가 아니라 다른 펭귄에 닿아도 시소에 앉은 것으로 처리됨. -> 떨어지는 펭귄이 다른 펭귄과 닿으면 펭귄이 밀리도록 설정함. 이를 통해 펭귄이 가운데로만 착지해 시소가 움직이지 않고 게임은 계속되는 현상을 방지함.
4) 소리파일을 불러올 때 절대 경로를 사용하던 것을 코드파일의 상대 위치를 기준으로 찾도록 수정함.
5) 소리 API는 사용 불가하다고 생각해 그 대신 게임의 몰입감과 현장을 높이기 위해 게임이 끝나면 세종과학기지와 장보고과학기지가 있는 위치의 기온과 풍속을 나타내도록 API를 사용함.
6) '공컴프 보드게임 수정' 파일에 소리 파일도 같이 넣어 게임이 진행될 때 소리가 나도록 함.
## 4. 실행 방법
이 프로젝트는 `uv` 패키지 매니저를 사용하여 실행 환경을 설정함. 아래 명령어를 순서대로 입력하면 실행할 수 있음.
```
1. uv가 없을 경우 uv 설치 (Windows 기준)
   powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
2. 게임 다운로드 및 실행
   git clone https://github.com/QIEclass2025/gcp.git
   cd gcp
   uv sync
   uv run penguin_seesaw.py
```
